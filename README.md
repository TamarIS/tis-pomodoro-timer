# FocusFlow: Premium Pomodoro Timer

A beautifully designed, simple, and intelligent Pomodoro timer to help maintain your focus and track your daily tasks. Built using purely native web technologies for maximum performance without the need to install a build pipeline or node.js.

## Features

- **Intelligent Timer**: Pre-configured standard focus lengths with middle-of-session interaction points.
- **Mid-Session Focus Check**: Halfway through your chosen focus time, an unobtrusive alert asks: "Are you on track?". If not, you have options to restart or jump right to a short break!
- **Task Management**: A simple to-do list allows you to track tasks. Data effortlessly saves to your browser automatically using Local Storage so it remains across sessions.
- **Adjustable Setup**: Standard settings included natively so you can increase your pomodoro times or break lengths directly in the UI.

## How to Run Locally

Since this app is built purely with HTML, CSS, and Vanilla JavaScript, there is no installation or backend server required. It runs natively and perfectly straight from your hard drive! 

### Instructions:

1. Open your web browser (Chrome, Safari, Firefox, Edge, etc.).
2. Navigate to your project folder using Finder.
3. Simply **double-click the `index.html` file** to open it.

*(Alternatively, you can drag and drop `index.html` directly into any open browser tab).*

That's it! Your Pomodoro Timer and Task list are now completely active and running. 

## Tech Stack

- CSS3 (Custom Glassmorphism, CSS Variables, Animations)
